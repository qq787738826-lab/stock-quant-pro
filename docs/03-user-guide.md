# 使用说明

1. 启动后先看“行情总览”的服务状态。
2. 在“智能选股”查看评分、买入区间、止损与目标位。
3. 在“回测中心”用至少一段完整牛熊周期测试，不能只看单次结果。
4. 在“持仓管理”生成委托；只有点击“人工确认”后状态才改变。
5. 最终下单必须在券商客户端再次核对代码、价格、数量和方向。
6. 免费行情可能延迟或中断。异常时软件使用演示数据，仅用于验证流程，不能据此实盘。

## 1.3.0 模拟交易流程

1. 在“智能选股”中对通过严格过滤的股票点击“交易计划”。
2. 进入“持仓管理”，核对买入区间、止损和目标价。
3. 点击“生成委托”，系统根据建议仓位自动计算100股整数倍数量。
4. 待确认委托只冻结模拟资金，不会直接成交。
5. 点击“确认成交”后，模拟账户自动扣款并建立持仓。
6. 当日新增数量不可卖，下一交易日自动按T+1规则解锁。
7. 点击“刷新行情并检查风险”，查看止损、止盈和移动止盈提示。
8. 点击持仓行“带入卖出”，核对数量和价格后创建并确认卖出委托。
9. 在模拟成交流水和权益曲线中复盘结果。
