# Change Log

## 1.2.0 - 2026-07-13

- 新增沪深主板股票列表同步和 PostgreSQL 本地证券库。
- 新增异步全市场扫描任务、任务进度、成功/失败统计。
- 新增 `market_scan_tasks`、`market_scan_results`、`market_sync_runs` 数据表。
- 扫描成功后自动保存日线数据、信号和交易计划。
- 新增数据管理中心、全市场行情库和全市场选股排行榜。
- 支持从候选结果直接进入 AI 动态分析和回测。
- Python 新增股票列表接口和批量动态分析接口。
- 后端行情读取取消模拟数据静默降级，优先读取 PostgreSQL 本地缓存。
- 版本统一升级为 1.2.0。
