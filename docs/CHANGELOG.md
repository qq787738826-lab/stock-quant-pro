# CHANGELOG

## 1.2.2 - 2026-07-13

- 扫描任务与结果按 `taskId` 绑定，默认展示最近一次正式全市场扫描。
- 新增扫描历史、扫描类型、正式结果标记、失败明细和失败股票重试。
- 新增行情增量更新任务，按最近90日数据进行主键覆盖入库。
- 新增严格候选过滤：评分、风险、成交额、ATR、20日涨幅、回撤、量比和涨停风险。
- 智能选股页面增加任务切换、严格候选开关、核心指标和过滤状态。
- 新增候选前N名批量历史回测，展示平均收益、平均胜率、平均回撤和个股明细。
- 数据管理中心增加扫描历史、行情更新进度和失败记录。
