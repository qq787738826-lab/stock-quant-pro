# Stock Quant Pro 1.3.1（A股短线量化终端）

面向 Windows 个人投资者的本地量化研究与模拟交易软件。当前版本覆盖沪深主板 A 股，默认排除创业板、科创板、北交所、ST 和退市股票，核心周期为 2–10 个交易日。

## 1.3.1 已完成

- 沪深主板股票列表、历史K线、增量更新和全市场异步扫描
- 扫描历史、失败重试、严格候选过滤和批量历史回测
- 候选股票一键加入交易计划
- 交易计划自动生成建议数量和待确认模拟委托
- 买入委托冻结模拟资金，卖出委托冻结可卖数量
- 人工确认后自动更新现金、持仓、成交费用与已实现盈亏
- 单股最大20%、最多5只持仓、重复委托和资金不足校验
- A股100股整数倍与T+1可卖数量模拟
- 止损、目标价、移动止盈风险检查
- 每个交易日16:30自动执行持仓行情刷新和风险检查
- 模拟成交流水、未处理风险事件和账户权益曲线
- 腾讯、新浪、东财和本地缓存的行情降级链路

## 技术栈

- Java 17、Spring Boot 3.2.8、Flyway
- PostgreSQL 16
- Python 3.11、FastAPI、AKShare、Pandas、NumPy
- Vue 3、TypeScript、Electron、ECharts

## 升级到1.3.1

保留旧项目的 `.env`，用1.3.1完整目录替换代码。在IDEA中重新加载Maven并启动Java，Flyway会自动执行：

```text
V4__simulated_trading.sql
```

随后启动Python和Electron客户端即可。

## 典型使用流程

```text
更新行情 → 全市场扫描 → 严格候选 → 加入交易计划
→ 生成待确认委托 → 人工确认模拟成交 → 持仓与风险检查
→ 模拟卖出 → 成交流水与权益曲线复盘
```

本系统只进行本地研究和模拟交易，不保存券商密码，也不会自动控制券商客户端。
