# Stock Quant Pro 1.3.0

## 核心升级

- 建立完整模拟账户资金与持仓闭环。
- 智能选股候选可直接加入交易计划。
- 交易计划可按建议仓位自动生成100股整数倍委托。
- 待确认买入冻结模拟资金，待确认卖出冻结可卖数量。
- 确认成交后自动计算佣金、税费、持仓成本和已实现盈亏。
- 支持单股20%、最多5只持仓、资金不足和重复委托检查。
- 支持T+1可卖数量、止损、目标价和移动止盈风险提示。
- 新增账户权益快照、权益曲线和模拟成交流水。

## 数据库升级

启动Java后，Flyway自动执行 `V4__simulated_trading.sql`。

## 安全边界

该版本仍然不连接券商账户，不自动下单。所有成交均为本地模拟成交。
