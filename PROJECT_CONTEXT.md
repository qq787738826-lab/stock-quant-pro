# Stock Quant Pro 项目上下文

- 当前版本：1.3.1
- Java：JDK 17、Spring Boot 3.2.8
- 数据库：PostgreSQL 16、Flyway
- Python：3.11、FastAPI、AKShare
- 前端：Vue 3、TypeScript、Electron
- 默认市场：沪深主板 A 股
- 排除范围：创业板、科创板、北交所、ST、退市股票
- 端口：Java 8080、Python 8001、Web 5173
- 交易方式：仅生成研究结果和人工确认委托，不自动控制券商客户端
- 最新完整代码基线：Stock Quant Pro 1.3.1 full

- 模拟交易：待确认委托、资金冻结、成交结算、T+1、持仓风控和权益曲线
